
1.下载nodejs并安装

[最新下载地址](https://nodejs.org/zh-cn/)

装完后输入命令行输入node -v
能够看到版本号则表示安装成功

2.安装yarn
命令行输入npm install -g yarn

3.启动服务
进入项目根目录双击start.bat
出现Listening at http://localhost:8080
则表示服务启动成功


## 本机访问：
浏览器输入 http://localhost:8080

## 局域网其他机器访问
请先保证服务器防火墙8080端口打开
然后 输入 http://{服务器IP地址}:8080
可以访问
